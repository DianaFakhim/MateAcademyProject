What is BEM?

BEM - BLOCK ELEMENT MODIFIER

BEM - is web-development methodology that lies in dividing interface into separate parts (blocks).

What problems BEM solves?

BEM simplifies code support in following:

  • Making code easier to understand
  • Helps avoid naming conflicts
  • Solves element specificity problem
  • Simplifies usability
